A qick walkthrough of the history of Loading Spinners



The Loading spinner or “Throbbers” are graphic control elements that indicate
that a computer program or application is performing an action in the background.






These background actions include, downloading contents, communication with
an external device or complex calculations.



This is different from the progress bar because it doe not indicate any specific increments
to show how much of an action has been completed.


The early versions of the loading spinners were called throbbers.
